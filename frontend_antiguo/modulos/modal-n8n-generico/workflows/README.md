# Workflows del modulo

## Politica
- `ACTIVO/` contiene un solo workflow.
- `VERSIONES/` guarda variantes anteriores del mismo modal.

## Flujo activo actual
- `ACTIVO/n8n-workflow-modal-generico-activo.json`
- Fuente activa real: `frontend/n8n-workflow-casetodo-simple-sin-ia.json`
- Build oficial: `scripts/build-n8n-casetodo-simple-sin-ia.mjs`
- Path webhook activo: `casetodo-simple`

## Versiones historicas registradas
- Web chat OpenAI cotizador (`casetodo-web-chat-v2`).
- Cotizacion completa (persona natural/empresa, IA cierre, PDF, SMTP, Telegram).
